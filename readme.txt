SIMPLE CONSOLE GRAPHICS is a simple personal project, created to explore and experiment with the fundamentals of computer graphics. 

- No dependencies, single file project.
- Platform independant, the only requirement being a c++ compiler.
- Object oriented design approach to math. (matrix2x1 * matrix2x2 really is a valid operation)
